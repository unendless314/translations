# Translation Guidelines for S01-E27

## 1. 翻譯目標與語氣
- **目標讀者**：對靈性、外星、形上學主題感興趣的中文讀者
- **整體語氣**：保持沉著、尊重的語氣，本集內容深度哲學性，需傳達智慧與洞見的氛圍
- **自然對話感**：使用流暢、自然的口語化中文，讓讀者感覺像在聽自然的對話
- **可讀性優先**：追求符合中文習慣的自然表達，而非逐字對照英文結構
- **哲學概念處理**：本集涉及大量形上學與存在論討論（如「四項法則」、「無摩擦存在」），需保持概念清晰，避免過度簡化或扭曲原意
- **錯誤修正**：若原文有明顯錯誤、遺漏或不完整之處，可依語境修正或補足，使譯文通順完整

## 2. 術語處理規範
- **權威來源**：所有專有名詞、靈性術語、族群名稱嚴格依照 `terminology.yaml` 的既定譯法
- **新詞處理**：遇到 terminology.yaml 未收錄的專業術語時：
  - 首次出現時使用「中文譯名（English Term）」格式標註
  - 後續出現僅使用中文譯名
  - 在翻譯 notes 欄位註記「新術語：[term] → [譯名]」以便後續收錄
- **音譯保留**：人名、地名若無慣用譯名，保持英文原文或採音譯，並在 notes 標註「待確認」

## 3. 翻譯風格與表達
- **忠實傳達意涵**：準確表達講者原意，保留第一人稱視角與情緒流動
- **避免逐字翻譯**：不要照搬英文語序或結構，優先考慮符合中文語境的自然說法
- **保留主觀語氣**：當講者表達信念、感受時，適當使用「我覺得」「我相信」「我感到」等詞彙
- **斷句靈活度**：若英文句子過長，可在逗點、頓號、連接詞處拆分，但避免刪減原文資訊
- **情緒詞使用**：可補充自然的口語語氣詞（如「嗯」「啊」「呢」），但保持克制，不宜浮誇

## 4. 格式與標記規範
- **音效與場景標記**：採用全形方括號【】置於句首（例：【音樂】【掌聲】【笑聲】）
- **數字與日期**：依原文含義翻譯（例：1990s → 1990 年代）

## 5. 人名與專名處理

### 5.1 本集主要人物

  - 首次提及：「伊舒瓦（Ishuwa）」
  - 後續可簡稱「伊舒瓦」或視語境使用「他」
- **Reuben Langdon（魯本·蘭登）**：節目主持人
  - 首次提及：「魯本·蘭登（Reuben Langdon）」
  - 後續可簡稱「魯本」、「主持人」或「他」
- **通靈管道（Channeler）**：最後片段出現
  - 可譯為「通靈者」、「管道」或「他」，視語境而定

### 5.2 外星族群名稱（嚴格依照 terminology.yaml）
- **Yahyel** → 亞耶爾
- **Yauaney** → 亞瓦尼（亞耶爾-人類混種文明）
- **Zeta Reticuli / Grays** → 網罟座 / 灰人

- 後續出現僅使用中文譯名

### 5.3 一般人名與地名
- **次要人物**（僅提及一兩次）：保留英文原文，在 notes 標註「人名保留英文」
- **地名**：使用慣用中文譯名（例：Phoenix, Arizona → 鳳凰城，亞利桑那州）
- **影視作品**：使用慣用譯名並加書名號（例：Avatar → 《阿凡達》）

## 6. 一致性要求
- **Topic 內部一致性**：同一 topic 的段落中，相同概念、人物、術語應使用一致的稱呼或譯法
- **全集一致性**：參照 terminology.yaml 確保整集術語使用統一
- **語氣連貫性**：同一講者的語氣風格應保持一致（例：某講者偏學術性 vs. 另一講者偏感性）

## 7. 翻譯欄位要求
每個段落需填寫：
- `text`：翻譯內容（必填，不可為空）
- `confidence`：high / medium / low（必填）
  - high：術語明確、語意清晰、無歧義
  - medium：有輕微不確定性，但不影響整體理解
  - low：原文模糊、術語待確認、或需後續審查
- `notes`：備註（可選）
  - 標註新術語、待確認的人名/地名
  - 說明翻譯選擇的理由
  - 標記需要審查的部分

## 8. 本集核心概念翻譯要點

### 8.1 四項法則（Four Laws）
本集核心教導，需保持概念清晰：
1. **You exist** → 你存在於此時此地
2. **The one is the all, and the all is the one** → 一即全，全即一
3. **What you put out is what you get back** → 你付出什麼，就會得到什麼
4. **Everything changes except the first three** → 一切都在變化，除了前三項法則

### 8.2 高頻形上學術語
- **actual nature** → 真實本質（本集極高頻，常搭配 "in alignment with"）
- **friction / friction-free** → 摩擦 / 無摩擦（核心概念，指無阻力、輕鬆流暢的存在狀態）
- **illusory beliefs** → 虛幻信念（與 actual nature 對立的概念）
- **unconditional love** → 無條件的愛（存在的本質）
- **vibration / vibratory reality** → 振動 / 振動實相
- **density level** → 密度層級（意識進化的層次，如第四、第五密度）
- **the one is the all, the all is the one** → 一即全，全即一（重複出現的核心概念）

### 8.3 抽象概念的處理
- **consciousness**（意識）：視語境可能指個人覺知、集體意識場、或萬物皆意識的哲學觀
- **existence**（存在）：常用於形上學討論，與「存有」(being) 互換
- **alignment**（對齊）：指與真實本質、高我、宇宙法則保持一致
- **co-creation**（共同創造）：人類與外星族群、或意識與實相的共同創造

## 9. 特殊處理原則
- **靈性概念保持中性**：對外星、通靈、轉世等主題保持客觀敘述，不加入評斷或刪減
- **哲學深度保持**：本集形上學討論深入，翻譯時避免過度簡化，保留思想的層次與深度
- **模糊內容**：若原文含義不明或資訊不足，保留合理程度的模糊性，並在 notes 標註原因
- **文化差異**：必要時可調整表達方式以符合中文讀者的理解習慣，但不改變核心意涵
- **重複性教導**：伊舒瓦的教導常以螺旋式重複深化，保留這種節奏，不過度精簡
