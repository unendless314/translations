# Translation Guidelines for S01-E24

## 本集特色與重點
本集訪談 Deete，一位來自鹽湖城的通靈者，探討她從嚴格摩門教背景轉變為靈媒與通靈管道的旅程。重點包括：
- 摩門教文化背景與宗教限制
- 爬蟲人族群（特別是天龍座α星系的 Balthazar）
- 維度 vs. 密度的概念區分
- 通靈方式（清醒通靈 vs. 深度出神）
- 歷史人物通靈（黛安娜王妃、楊百翰）

## 1. 翻譯目標與語氣
- **目標讀者**：對靈性、外星、形上學主題感興趣的中文讀者
- **整體語氣**：保持紀錄片的沉著、尊重語氣，避免戲謔或過度口語化
- **自然對話感**：使用流暢、自然的口語化中文，讓讀者感覺像在聽自然的對話
- **可讀性優先**：追求符合中文習慣的自然表達，而非逐字對照英文結構
- **錯誤修正**：若原文有明顯錯誤、遺漏或不完整之處，可依語境修正或補足，使譯文通順完整

## 2. 術語處理規範
- **權威來源**：所有專有名詞、靈性術語、族群名稱嚴格依照 `terminology.yaml` 的既定譯法
- **新詞處理**：遇到 terminology.yaml 未收錄的專業術語時：
  - 首次出現時使用「中文譯名（English Term）」格式標註
  - 後續出現僅使用中文譯名
  - 在翻譯 notes 欄位註記「新術語：[term] → [譯名]」以便後續收錄
- **音譯保留**：人名、地名若無慣用譯名，保持英文原文或採音譯，並在 notes 標註「待確認」

### 本集特定術語注意事項

**維度 vs. 密度**（重要概念區分）：
- **Dimension（維度）**：物理空間的維度，或靈性層面的存在層級
  - 例：third dimension（第三維度）、fourth dimension（第四維度）
- **Density（密度）**：意識振動層次的分級，與物理維度不同
  - 例：third density（第三密度）、fourth density（第四密度）
- **翻譯要求**：必須嚴格區分「維度」與「密度」，不可混用
- 本集中 Deete 與 Reuben 特別討論了這兩個概念的差異

**爬蟲人相關術語**：
- **Reptilian（單數）** → 爬蟲人（泛指）
- **Reptilians（複數）** → 爬蟲人（族群）
- **Draconis** → 天龍座人（特定族群）
- **Alpha Draconis** → 天龍座α星系
- **Balthazar** → 巴爾薩澤（Deete 的指導靈，來自天龍座α）
- 本集強調並非所有爬蟲人都是負面的，翻譯時避免帶入負面暗示

**摩門教相關術語**：
- **Mormon** → 摩門教徒 / 摩門教的
- **Mormon church** → 摩門教會
- **LDS / Latter-Day Saints** → 後期聖徒（摩門教正式名稱）
- **FLDS** → 摩門教基本教義派（分離教派）
- **Salt Lake City** → 鹽湖城
- **Brigham Young** → 楊百翰（使用慣用譯名）
- **Joseph Smith** → 約瑟·斯密（使用慣用譯名）

**通靈方式術語**：
- **channeling** → 通靈（動詞/動名詞）
- **channeler** → 通靈者 / 通靈管道
- **medium** → 靈媒（特指與已逝者溝通）
- **trance** → 出神狀態
- **deeper trance** → 深度出神
- **translating in real time** → 即時翻譯（Deete 的清醒通靈方式）
- Deete 強調她採用「清醒通靈」而非深度出神，翻譯時需注意此區別

**外星族群術語**：
- **Gray / Zeta** → 小灰人 / 澤塔族（同義）
- **Mantis** → 螳螂族
- **Lyran** → 天琴座人
- **Council of Beings** → 存有議會
- **Council of Light** → 光之議會（Deete 的療癒集體）

## 3. 翻譯風格與表達
- **忠實傳達意涵**：準確表達講者原意，保留第一人稱視角與情緒流動
- **避免逐字翻譯**：不要照搬英文語序或結構，優先考慮符合中文語境的自然說法
- **保留主觀語氣**：當講者表達信念、感受時，適當使用「我覺得」「我相信」「我感到」等詞彙
- **斷句靈活度**：若英文句子過長，可在逗點、頓號、連接詞處拆分，或在保留原文語義的前提下去除贅詞
- **情緒詞使用**：可補充自然的口語語氣詞（如「嗯」「啊」「呢」），但保持克制，不宜浮誇

## 4. 格式與標記規範
- **音效與場景標記**：採用全形方括號【】置於句首（例：【音樂】【掌聲】【笑聲】）
- **數字與日期**：依原文含義翻譯（例：1990s → 1990 年代）

## 5. 人名與專名處理
- **主要人物**（反覆出現的主持人、受訪者）：
  - 首次出現使用「音譯（English Name）」格式
  - 例：魯本·蘭登（Reuben Langdon）、迪蒂（Deete）
  - 後續出現可視流暢度選擇：僅用音譯、或用角色稱謂（主持人、受訪者）
- **次要人物**（僅提及一兩次）：
  - 保留英文原文
  - 在 notes 標註「人名保留英文」
- **地名**：
  - 使用慣用中文譯名（例：Salt Lake City → 鹽湖城；Phoenix, Arizona → 鳳凰城，亞利桑那州）
  - 不確定的地名保留英文，在 notes 標註「地名待確認」
- **場所/組織名稱**：
  - 首次出現使用「中文譯名（English Name）」格式
  - 例：飛碟咖啡館（Flying Saucer Cafe）
- **影視作品**：
  - 使用慣用譯名並加書名號（例：Avatar → 《阿凡達》）
  - 若無慣用譯名：《原文》或 中文譯名《原文》

- **Balthazar** → 巴爾薩澤（爬蟲人指導靈，首次出現標註來源）
- **Princess Diana** → 黛安娜王妃（使用慣用譯名）
- **Brigham Young** → 楊百翰（摩門教領袖，使用慣用譯名）

- **Rob Gauthier** → 羅伯·高提耶（通靈者，首次出現標註）

## 6. 一致性要求
- **Topic 內部一致性**：同一 topic 的段落中，相同概念、人物、術語應使用一致的稱呼或譯法
- **全集一致性**：參照 terminology.yaml 確保整集術語使用統一
- **語氣連貫性**：同一講者的語氣風格應保持一致（例：某講者偏學術性 vs. 另一講者偏感性）

## 7. 翻譯欄位要求
每個段落需填寫：
- `text`：翻譯內容（必填，不可為空）
- `confidence`：high / medium / low（必填）
  - high：術語明確、語意清晰、無歧義
  - medium：有輕微不確定性，但不影響整體理解
  - low：原文模糊、術語待確認、或需後續審查
- `notes`：備註（可選）
  - 標註新術語、待確認的人名/地名
  - 說明翻譯選擇的理由
  - 標記需要審查的部分

## 8. 特殊處理原則
- **靈性概念保持中性**：對外星、通靈、轉世等主題保持客觀敘述，不加入評斷或刪減
- **模糊內容**：若原文含義不明或資訊不足，保留合理程度的模糊性，並在 notes 標註原因
- **文化差異**：必要時可調整表達方式以符合中文讀者的理解習慣，但不改變核心意涵

### 本集特定文化背景處理

**摩門教文化背景**：
- 翻譯涉及摩門教的內容時，保持客觀中立，既不美化也不貶抑
- Deete 描述的宗教壓抑經驗是她個人經歷，翻譯時忠實呈現其感受
- 一夫多妻（polygamy）、FLDS 分支等敏感話題需如實翻譯，避免迴避
- 對楊百翰、約瑟·斯密等歷史人物的通靈描述，作為靈性訊息呈現，不涉入歷史真實性評判

**爬蟲人正面化議題**：
- 本集重要訊息之一是「並非所有爬蟲人都是負面的」
- 翻譯時避免使用帶有負面暗示的詞彙（如「邪惡的」、「恐怖的」）
- Balthazar 作為「指導靈」與「保護者」，應以正面角色呈現
- 保留 Deete 對「黑暗能量」的描述，但理解這是指頻率層次而非道德評判

**通靈真實性**：
- 對於 Deete 的主觀經驗（如「我看見」、「我感覺到」），忠實翻譯，不質疑也不強化
- 通靈內容作為「她接收到的訊息」呈現，讀者自行判斷
- 對於「logical mind」（邏輯心智）質疑通靈真實性的段落，保留這種內在掙扎的描述

**維度密度的科學性**：
- Deete 嘗試用科學術語（X,Y,Z, spacetime）解釋維度概念
- 雖然這種解釋可能不符合主流物理學，仍需忠實翻譯她的理解框架
- 不在翻譯中加入「所謂的」、「自稱的」等質疑詞彙
