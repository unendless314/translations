# Translation Guidelines for S01-E25

## 1. 翻譯目標與語氣
- **目標讀者**：對靈性、外星、形上學主題感興趣的中文讀者
- **本集特性**：本集為獵戶座光之議會的通靈訪談，探討人類起源、阿努納奇創造、外星DNA混合、階層系統等深層宇宙歷史
- **訪談形式**：主持人魯本·蘭登與女性通靈者的問答互動，通靈者傳遞來自議會女性存有、昆蟲族、天龍座存有的訊息
- **整體語氣**：保持訪談的探索性與尊重語氣，傳達複雜的宇宙歷史與靈性概念時兼顧深度與可讀性
- **概念複雜度**：本集涉及時間線、多種族DNA、業力循環、維度差異等抽象概念，翻譯時需確保邏輯清晰
- **自然對話感**：使用流暢、自然的口語化中文，讓讀者感覺像在聽自然的對話
- **可讀性優先**：追求符合中文習慣的自然表達，而非逐字對照英文結構
- **錯誤修正**：若原文有明顯錯誤、遺漏或不完整之處，可依語境修正或補足，使譯文通順完整

## 2. 術語處理規範
- **權威來源**：所有專有名詞、靈性術語、族群名稱嚴格依照 `terminology.yaml` 的既定譯法
- **新詞處理**：遇到 terminology.yaml 未收錄的專業術語時：
  - 首次出現時使用「中文譯名（English Term）」格式標註
  - 後續出現僅使用中文譯名
  - 在翻譯 notes 欄位註記「新術語：[term] → [譯名]」以便後續收錄
- **音譯保留**：人名、地名若無慣用譯名，保持英文原文或採音譯，並在 notes 標註「待確認」

## 3. 翻譯風格與表達
- **忠實傳達意涵**：準確表達講者原意，保留第一人稱視角與情緒流動
- **避免逐字翻譯**：不要照搬英文語序或結構，優先考慮符合中文語境的自然說法
- **保留主觀語氣**：當講者表達信念、感受時，適當使用「我覺得」「我相信」「我感到」等詞彙
- **斷句靈活度**：若英文句子過長，可在逗點、頓號、連接詞處拆分，或在保留原文語義的前提下去除贅詞
- **情緒詞使用**：可補充自然的口語語氣詞（如「嗯」「啊」「呢」），但保持克制，不宜浮誇

## 4. 格式與標記規範
- **音效與場景標記**：採用全形方括號【】置於句首（例：【音樂】【掌聲】【笑聲】）
- **數字與日期**：依原文含義翻譯（例：1990s → 1990 年代）

## 5. 人名與專名處理
- **主要人物**（反覆出現的主持人、受訪者）：
  - 首次出現使用「音譯（English Name）」格式
  - 例：魯本·蘭登（Reuben Langdon）、布莉姬·尼爾森（Bridget Nielsen）
  - 後續出現可視流暢度選擇：僅用音譯、或用角色稱謂（主持人、受訪者）
- **次要人物**（僅提及一兩次）：
  - 保留英文原文
  - 在 notes 標註「人名保留英文」
- **地名**：
  - 使用慣用中文譯名（例：Sedona, Arizona → 賽多納，亞利桑那州；Santa Monica → 聖莫尼卡）
  - 不確定的地名保留英文，在 notes 標註「地名待確認」
- **場所/組織名稱**：
  - 首次出現使用「中文譯名（English Name）」格式
  - 例：飛碟咖啡館（Flying Saucer Cafe）
- **影視作品**：
  - 使用慣用譯名並加書名號（例：Avatar → 《阿凡達》）
  - 若無慣用譯名：《原文》或 中文譯名《原文》

## 6. 一致性要求
- **Topic 內部一致性**：同一 topic 的段落中，相同概念、人物、術語應使用一致的稱呼或譯法
- **全集一致性**：參照 terminology.yaml 確保整集術語使用統一
- **語氣連貫性**：同一講者的語氣風格應保持一致（例：某講者偏學術性 vs. 另一講者偏感性）

## 7. 翻譯欄位要求
每個段落需填寫：
- `text`：翻譯內容（必填，不可為空）
- `confidence`：high / medium / low（必填）
  - high：術語明確、語意清晰、無歧義
  - medium：有輕微不確定性，但不影響整體理解
  - low：原文模糊、術語待確認、或需後續審查
- `notes`：備註（可選）
  - 標註新術語、待確認的人名/地名
  - 說明翻譯選擇的理由
  - 標記需要審查的部分

## 8. 特殊處理原則
- **靈性概念保持中性**：對外星、通靈、轉世等主題保持客觀敘述，不加入評斷或刪減
- **模糊內容**：若原文含義不明或資訊不足，保留合理程度的模糊性，並在 notes 標註原因
- **文化差異**：必要時可調整表達方式以符合中文讀者的理解習慣，但不改變核心意涵
- **時間線複雜性**：本集涉及多個時間層次（亞特蘭提斯、大洪水、阿努納奇創造、恐龍時代等），翻譯時需保持時序邏輯清晰
- **多種族概念**：阿努納奇被描述為多種族聯盟而非單一種族，翻譯時需傳達此複雜性

## 9. S01-E25 特定術語與指引

### 9.1 核心概念與族群
- **Orion Council of Light / 獵戶座光之議會**：本集通靈存有所代表的靈性議會
- **Anunnaki / 阿努納奇**：創造現代人類的外星族群聯盟
  - **關鍵理解**：本集強調阿努納奇並非單一種族，而是來自獵戶座區域的多種族聯盟
  - 包含：昴宿星人（Pleiadian）、爬蟲人（Reptilian）、昆蟲族（Insectoid）等
  - 初衷：建立地球統治權，後因業力循環演化為人類的指導靈
- **Atlantis / 亞特蘭提斯**：第五維度、第四密度的遠古文明
  - **與現代人類的區別**：亞特蘭提斯人與阿努納奇創造的現代人類是不同物種，存在DNA斷層
  - 毀滅原因：意識快速擴展與實驗失控，導致化學戰爭與地球自然重置
- **diluvian / the great flood / 大洪水**：阿努納奇為重置失控的人類實驗而刻意製造的洪水
  - **與亞特蘭提斯洪水的區別**：亞特蘭提斯的毀滅是自然地球變動，大洪水是阿努納奇的刻意行為

### 9.2 外星族群特質
- **Reptilian / Reptoids / 爬蟲人、爬蟲族**：
  - DNA 貢獻：生存技能、適應惡劣環境的能力
  - 意識特質：階層式統治、缺乏同情心的支配傾向
  - **重要澄清**：本集指出「邪惡爬蟲人」是簡化的刻板印象，階層系統源自獵戶座而非僅爬蟲族
- **Insectoid / 昆蟲族**：
  - DNA 貢獻：直覺能力、同理心、基因工程專業
  - 協助阿努納奇構建人類基因組
- **Pleiadian / 昴宿星人**：
  - 常被誤解為純粹女性、柔和能量
  - **本集澄清**：昴宿星人也理解階層結構，男性角色更像導師教師
- **serpent species / 蛇族**：
  - 來自「蛇星系統」（包含天龍座α星）的廣義族群
  - 不限於爬蟲形態，也包含哺乳類、鳥類形態
  - 部分個體選擇統治意識路徑，影響地球人類意識
- **Zeta / 齊塔人**：
  - 無法繁殖的集體意識族群，缺乏情感與個體性
  - 混種計畫目的：透過人類 DNA 理解情感、喜悅、個體性
  - 交換：提供科技知識，獲得情感理解與繁殖方案

### 9.3 維度與密度概念
- **third density / 第三密度**：人類當前的意識層次與物質實相
- **5th dimensional beings / 第五維度存有**：亞特蘭提斯時期的類人生物
- **dimension vs. density**：
  - dimension（維度）：存在的空間層級
  - density（密度）：意識的進化程度
  - 例：「fifth dimension, fourth density」（第五維度，第四密度）

### 9.4 靈性與歷史事件
- **Christ consciousness / 基督意識**：靈性門戶關閉前的「最後希望」，提醒人類自身力量
- **Buddha energy / 佛陀能量**：早期引入的靈性干預，帶來內在平靜
- **2012**：新門戶開啟的起點，人類覺醒與重新建立接觸的關鍵時刻
- **Roman era / 羅馬時代**：開放的物理性外星接觸終止的時期
  - 原因：人類集體意識選擇物質主義，崇拜「無益的神祇」，實際上是要求光之存有離開

### 9.5 古代遺址
- **Gobekli Tepe / 哥貝克力石陣**：非常古老的「促進化合物」或直接通訊轉錄裝置
- **Giza pyramid / 吉薩金字塔**：能量門戶或星門，而非通訊裝置
- **Nazca / 納斯卡**：三指生物協助阿努納奇建立傳送裝置的區域
- **Antarctica / 南極洲**：受保護的隔離區，通往地心家族的門戶，「即將揭露」的重大歷史

### 9.6 關鍵術語對照
- **hierarchical system / 階層系統**：非僅爬蟲族特質，源自獵戶座能量，關鍵在於避免「高低」轉為「支配」
- **Reptilian agenda / 爬蟲人計畫**：統治與支配的意識模式
- **rulingship / 統治權**：透過男性血脈傳承王權，源自獵戶座父權能量
- **Abrahamic covenant / 亞伯拉罕盟約**：男性主導的宗教契約，反映獵戶座父權結構
- **karmic cycle / 業力循環**：阿努納奇創造人類後因大洪水重置產生的重複業力模式
- **evolutionary experiment / 進化實驗**：人類作為多種族 DNA 混合的宏大實驗
- **enslavement programs / 奴役程式**：第三密度中限制意識的系統性機制
- **The Matrix / 母體**：限制意識的系統性框架，引用《駭客任務》概念
- **matrix-style living / 母體式生活**：第三密度的受限存在模式

### 9.7 人名處理
- **Reuben Langdon / 魯本·蘭登**：本集主持人
  - 首次出現：魯本·蘭登（Reuben Langdon）
  - 後續可簡稱「主持人」或「魯本」
- **通靈者**：本集女性通靈者未具名提及，稱為「她」、「通靈者」、「這位女士」等

### 9.8 翻譯注意事項
- **"She says" 處理**：大量出現「她說」引導通靈訊息，可視流暢度省略部分重複的「她說」
- **視覺描述**：通靈者常說「她在顯示給我看」「我看到」，保留這些描述以傳達通靈過程的視覺性質
- **時間線模糊**：存有常表達「線性時間難以描述」「時間線複雜」，保留此模糊性並在 notes 標註
- **多義詞澄清**：
  - "human" 可能指現代人類或廣義類人生物，需依語境判斷
  - "flood" 可能指亞特蘭提斯洪水或阿努納奇大洪水，需區分
  - "consciousness takeover" 指意識被接管，不同於 DNA 改造
